# UI/UX Pro Max Skill

## Overview

**UI/UX Pro Max Skill** is a powerful AI skill that provides professional design intelligence for building user interfaces and experiences across multiple platforms. With over 41,500 stars on GitHub, it's a trusted resource for developers and designers.

## Features

- **Multi-Platform UI/UX Design** - Build consistent interfaces across web, mobile, and desktop
- **Modern Framework Support** - Integrated with React, TailwindCSS, and HTML5
- **Design Intelligence** - AI-powered design pattern recognition and suggestions
- **Accessibility Compliance** - Built-in WCAG guidelines and accessibility features
- **Responsive Design** - Mobile-first, responsive design patterns
- **Component Libraries** - Comprehensive reusable UI component collections

## Supported Environments

- **Claude Code** - Primary AI coding assistant
- **OpenAI Codex** - Code completion and generation
- **Cursor & Windsurf** - AI-enhanced editors
- **OpenClaw** - Autonomous agent framework
