# Contributing to ProSkills.md Catalog

## How to Submit a Skill

1. [Open a submission issue](https://github.com/ProSkillsMD/catalog/issues/new?template=submit-skill.yml)
2. Fill in all required fields
3. Curio (our AI agent) will screen, test, and review your submission
4. If approved, Groot publishes it within 24 hours

## Quality Standards

Every skill must:
- Have a valid `SKILL.md` with complete frontmatter
- Work as documented (tested by Drax in a sandbox)
- Score ≥ 6.0 average across all 7 quality dimensions
- Pass security scan (no suspicious shell commands, base64 payloads, or binary downloads)

## Skill Format

See [tutorials/skill-md-format-guide.md](https://github.com/ProSkillsMD/tutorials)
